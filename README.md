<h1 align="center">H4sh</h1>
<p align="center">A utility to create hash signatures from strings.</p>
<p align="center">Written in Golang</p>

---

## To run.
> go run h4sh.go
## To build.
Create a compiled file.
> go build h4sh.go

### Usage:
``` 
     -H4sh-
  Usage Method:
> md5 "string"
> sha1 "string"
> sha265 "string"
> sha512 "string"
```
### Command
`$ h4sh md5 "string"`
### Output
```
MD5 <- string
md5: b45cffe084dd3d20d928bee85e7b0f21
```
